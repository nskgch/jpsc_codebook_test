Sheet number¶

Variable name: REC¶

Description¶

It is the index attached to a data file. For each survey, the data is divided into five files, and the name of each data file is a combination of prefix indicating the survey number and a data file index.

For example, for data file p10_1csv, REC is 1; for data file p10_2.csv, REC is 2.
